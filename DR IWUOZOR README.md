## Project Directory

### 1. Simple Calculator (`Project 1/`)
* **Description:** A functional arithmetic tool that handles addition, subtraction, multiplication, and division.
* **Key Features:** User input validation and error handling for division by zero.
* **How to Run:** Run `python "Project 1/calculator.py"`

### 2. Number Guessing Game (`Project 2/`)
* **Description:** An interactive game where the computer selects a random integer and the user must find it using logic.
* **Key Features:** Uses the `random` library and `while` loops for continuous play.
* **How to Run:** Run `python "Project 2/guessing_game.py"`

### 3. Word Counter Tool (`Project 3/`)
* **Description:** A text analysis utility that calculates the total number of words in any given string.
* **Key Features:** Utilizes string manipulation and the `.split()` method.
* **How to Run:** Run `python "Project 3/word_counter.py"`

### 4. Task Manager (To-Do List) (`Project 4/`)
* **Description:** A console-based application to manage daily tasks.
* **Key Features:** Allows for adding, viewing, and organizing tasks dynamically.
* **How to Run:** Run `python "Project 4/todo_list.py"`
